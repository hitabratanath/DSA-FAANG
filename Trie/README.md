# Trie Problems for FAANG Interview Preparation

## Problem Categories

### Basic Implementation
1. **01_implement_trie.py** - Implement Trie (Leetcode 208)
2. **02_implement_trie_2.py** - Implement Trie-2 with Count Functions

### String Problems  
1. **01_longest_word_all_prefixes.py** - Longest Word With All Prefixes
2. **02_distinct_substrings.py** - Number of Distinct Substrings
3. **03_word_search_ii.py** - Word Search II (Leetcode 212) ⭐ High Priority
4. **04_add_search_word.py** - Add and Search Word (Leetcode 211) ⭐ High Priority
5. **05_replace_words.py** - Replace Words (Leetcode 648)

### Bit Manipulation
1. **01_maximum_xor_two_numbers.py** - Maximum XOR of Two Numbers (Leetcode 421)
2. **02_maximum_xor_with_element.py** - Maximum XOR With Element From Array

## Study Order
1. **Start with Basic Implementation** - Master Trie structure first
2. **Move to String Problems** - Apply Trie to string processing
3. **Finish with Bit Manipulation** - Advanced Trie applications

## FAANG Priority
- ⭐ **High Priority**: Problems 3, 4 from String Problems
- 🔥 **Google Favorites**: Word Search II, Maximum XOR problems
- 💡 **Amazon Favorites**: Add and Search Word, Replace Words

## Resources
- **YouTube Playlist**: Striver's Trie Series
- **LeetCode Practice**: Individual problem links
- **Time Complexity**: Most Trie operations are O(L) where L is word length

## Interview Tips
- Always explain Trie structure before coding
- Mention space-time tradeoffs vs HashMap
- Show both recursive and iterative approaches when possible
